# Freelance Dashboard (React + TypeScript)

A type-safe freelance management dashboard to track clients, projects, and payments.
live deployed link: https://freelance-management-dashboard.vercel.app/

## Technologies Used
- React + TypeScript
- Context API + useReducer
- Tailwind CSS (styling)
- Vite (build tool)

## Main Features
- Fully typed data models (Client, Project, Payment)
- Global state with discriminated unions
- Type-safe components and props
- Search, filter, and mark projects as paid
- Dashboard statistics
- Safe client lookup with type narrowing
